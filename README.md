# 📵 자동 응답 앱 (AutoReply)

전화번호부에서 선택한 대상 번호로부터 전화가 오면 자동으로 거절하고 자동응답 문자를 발송하는 안드로이드 앱입니다.

## 🚀 빌드 방법 (GitHub Actions)

1. 이 저장소의 모든 파일을 본인의 GitHub Repository에 업로드
2. 업로드 즉시 GitHub Actions가 자동으로 빌드 시작
3. **Actions 탭 → 최신 빌드 클릭 → Artifacts → AutoReply-APK 다운로드**
4. 압축 풀고 .apk 파일을 갤럭시로 옮긴 후 설치

## 📱 갤럭시 설치 방법

1. 설정 → 보안 및 개인정보 보호 → **알 수 없는 출처에서 앱 설치 허용**
2. .apk 파일 실행 → 설치
3. 앱 실행 후 권한 모두 허용 (연락처, 전화, 문자)

## 🎯 사용법

1. **기능 활성화** 스위치 ON
2. **자동 응답 메시지** 작성 후 저장
3. **+ 추가** 버튼으로 대상 번호 선택
4. 끝! 등록된 번호에서 전화 오면 자동 거절 + 문자 발송

## 🔒 권한 안내

- **연락처**: 전화번호부에서 대상 선택
- **전화 상태/응답**: 수신 전화 거절
- **SMS 발송**: 자동 응답 문자 전송
